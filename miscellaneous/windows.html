Pro Tips & Tricks

Taskbar Tricks

  • Shift + Click on taskbar icon - Open new instance
  • Ctrl + Click on taskbar icon - Cycle through windows
  • Middle Click on taskbar icon - Open new instance
  • Shift + Right-Click on taskbar - Show window menu

File Explorer Secrets

  • Alt + D - Focus address bar instantly
  • F4 in address bar - Show recent locations dropdown
  • Type cmd in address bar - Open Command Prompt in that folder
  • Type powershell in address bar - Open PowerShell in that folder
  • Shift + Right-Click on folder - Extended menu with “Copy as path”

Run Dialog Tricks

  • Ctrl + Shift + Enter in Run Dialog - Run as Administrator
  • Type \computer-name - Connect to network computer
  • Type website URL - Opens in default browser

Multi-Monitor Magic

  • Win + Shift + Left/Right Arrow - Move window between monitors
  • Win + P - Project/display options (duplicate, extend, etc.)

Quick Copy Tricks

  • Ctrl + Shift + C/V - Copy/paste in Command Prompt
  • Win + V - Open clipboard history (enable in Settings first)
  • Select file in Explorer, Ctrl + Shift + C - Copy file path
TipProductivity Hack

Pin frequently used apps to taskbar and use Win + [1-9] to launch them instantly!

Safety Notes & Best Practices

WarningRegistry Editor Warning

Only modify the registry if you know exactly what you are doing. Always create a backup before making changes. Incorrect registry edits can make Windows unstable or unbootable.

WarningAdministrator Commands

Commands like sfc /scannow, chkdsk, and DISM require administrator privileges. Right-click Command Prompt/PowerShell and select “Run as administrator”.

WarningMaintenance Time

sfc /scannow takes 15-30 minutes. chkdsk and DISM can take 30+ minutes. Run these when you have time and will not need your computer.

Safe Practices

  • Lock your PC when leaving: Win + L
  • Regularly update Windows via Settings > Windows Update
  • Enable BitLocker for drive encryption (if available)
  • Use strong passwords and Windows Hello if possible
  • Keep important files backed up to cloud or external drive
  • Run Windows Defender scans regularly
  • Only download software from trusted sources

Maintenance Schedule

NoteRecommended Schedule
  • Monthly: Run Disk Cleanup (cleanmgr), check for Windows updates
  • Quarterly: Run sfc /scannow, review installed programs
  • Yearly: Full system backup, DISM health check
